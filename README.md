# Assembly Playground
 
A javascript-based Assembly emulator.
 
## Controls
 
- ``RUN`` run the full program (instant or with a step delay via the interval slider)
- ``STEP`` execute one instruction at a time
- ``STOP`` reset everything, log, registers and flags to 0

Check out the documentation page [here](https://mittnamnstartarintemedett-e.github.io/slutprojekt-webutveckling/documentation.html)
