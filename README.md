# Assembly Playground
 
A javascript-based Assembly emulator.
 
## Controls
 
- ``RUN`` run the full program, either instantly or in intervals using the slider
- ``STEP`` execute one instruction at a time
- ``STOP`` stops execution, resets everything, log, registers and flags to 0
- ``DOCUMENTATION`` brings you to the documentation page
-
